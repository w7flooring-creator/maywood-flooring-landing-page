## Code fixes in this branch

- Add descriptive screen-reader context to both generic `READ MORE` links. A local mobile Lighthouse rerun now scores SEO 100 and passes the link-text audit.
- Give the Silent Foundation image intrinsic dimensions. The rerun passes the unsized-images audit with CLS 0.001.
- Add smaller responsive hero image candidates and responsive logo variants.
- Replace borderline `#777` category text with the design system's accessible muted foreground token.
- Make collection-store titles and descriptions distinct from their marketing landing pages.
- Add collection and flooring type context to product SEO fallbacks so same-name products remain distinguishable.
- Collapse whitespace and cap generated meta descriptions at 160 characters.
- Redirect the production apex hostname to the canonical `www` hostname with HTTP 308 while leaving `workers.dev` and preview hostnames untouched.

The SEO fallback also rejects both short SEO descriptions and short category descriptions, preventing future placeholder values from becoming production metadata.
